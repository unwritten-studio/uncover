## Schreiben & Speichern

Wer Kapitel speichern möchte, braucht einmalig einen GitHub Access Token:

1. [github.com/settings/personal-access-tokens/new](https://github.com/settings/personal-access-tokens/new) öffnen
2. **Repository access** → nur `uncover` · **Contents** → Read and write
3. Token kopieren → oben rechts im Dashboard auf **⚙ Token** klicken → einfügen

Der Token wird im Browser gespeichert und muss nicht erneut eingegeben werden.
